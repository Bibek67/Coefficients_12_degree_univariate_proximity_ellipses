# Coefficients_12_degree_univariate_proximity_ellipses

This project contains all 13 coefficients of the 12-degree univariate in the variable "x1" resulting from the proximity of two non-intersecting coplanar ellipses. The 12-degree univariate is defined as: s(x1) = t1 x1^12 + t2 x1^11 + ... + t13.
